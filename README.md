# Image-Editor
Beta test version, may include bugs

What's needed? 
Python 3.10
Libraries PyQt5 and PIL(pillow)

What it can do?
Convert image to black and white color scheme
Change contrast of the image
Rotate the image
Blur the image
Flip sides of the image 
Convert .webp, .png. and more to .jpg
